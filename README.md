# MedCore - Sistema de Gestão Hospitalar

## Objetivo

O MedCore é um sistema web desenvolvido para auxiliar clínicas e consultórios médicos no gerenciamento de pacientes, consultas e atendimentos.

## Tecnologias Utilizadas

* Python 3
* Django
* SQLite
* HTML
* CSS
* JavaScript

## Funcionalidades

### Pacientes

* Cadastro
* Edição
* Exclusão
* Listagem

### Médicos

* Cadastro
* Edição
* Exclusão
* Listagem

### Consultas

* Agendamento
* Controle de status

### Atendimentos

* Registro de prontuário
* Histórico de atendimentos

### Secretárias

* Cadastro
* Vinculação a médicos

### Usuários

* Login
* Perfil
* Alteração de senha

## Banco de Dados

O sistema utiliza SQLite para armazenamento das informações.

## Arquitetura

O projeto foi desenvolvido utilizando o padrão MTV (Model-Template-View) do framework Django.

## Conclusão

O sistema atende aos requisitos propostos e apresenta funcionalidades adicionais que ampliam sua utilização em ambientes clínicos.
